PhyloCanvas

This is a demonstration of the HTML5 Tree-drawing widget PhyloCanvas

PhyloCanvas is written entirely in browser-native JavaScript, using no external libraries, and uses the HTML5 Canvas element

How to use

Initializing the Viewer

You only need 2 lines of code to get your phylocanvas instance up and running. One to initialize the viewer and a second to load the file. 

var phylocanvas = new PhyloCanvas.Tree(div_name); // initialize your tree viewer

phylocanvas.load(newick_file); // load your tree file

API - Tree

.load(filename,[tree name])

FAQs

Why Canvas and not SVG?

Both offer significant advantages, but ultimately at the start of the project (2011) the Canvas API was better supported across browers than inline SVG
